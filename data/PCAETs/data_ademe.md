# Description de la page Open Data
L’ADEME met à disposition l’**ensemble des données saisies sur Territoires&Climat** en open data. Plusieurs jeux de données sont ainsi disponibles sur cette page, ainsi que sur le portail Open Data de l’ADEME, qui permet une meilleure réutilisation (notamment via API). Tous les jeux de données sont actualisés quotidiennement.


Afin de faciliter l’exploitation des données, les jeux de données sont présentés sur cette page, et la description détaillée du contenu de chaque fichier est disponible dans fichier « Notice d’exploitation des données » ci-dessous.

from: https://www.territoires-climat.ademe.fr/opendata#_blank
